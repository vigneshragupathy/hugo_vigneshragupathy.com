# Hugo Website - vigneshragupathy.com

This repository contains the source code for my personal website built with Hugo.

## How to Run

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/vigneshragupathy/hugo_vigneshragupathy.com.git
    cd hugo_vigneshragupathy.com
    ```

2.  **Run the Hugo server:**

    To start the local development server and bind it to all interfaces (useful for accessing from other devices or within containers):

    ```bash
    hugo server --bind 0.0.0.0
    ```

## Formatting Tips

You can use HTML `<span>` tags with inline styles to add color and highlights to your text in Markdown files.

**Examples:**

*   **Purple Text:**
    ```html
    <span style="color:#6366f1">**I decided to build this without writing a single line of manual code.**</span>
    ```
    Result: <span style="color:#6366f1">**I decided to build this without writing a single line of manual code.**</span>

*   **Orange Text:**
    ```html
    <span style="color:#ea580c">**I decided to build this without writing a single line of manual code.**</span>
    ```
    Result: <span style="color:#ea580c">**I decided to build this without writing a single line of manual code.**</span>

*   **Yellow Background with Black Text:**
    ```html
    <span style="background-color: #fef08a; color: #000000; padding: 0 4px;">**I decided to build this without writing a single line of manual code.**</span>
    ```
    Result: <span style="background-color: #fef08a; color: #000000; padding: 0 4px;">**I decided to build this without writing a single line of manual code.**</span>

*   **Teal Text:**
    ```html
    <span style="color:#0d9488">**I decided to build this without writing a single line of manual code.**</span>
    ```
    Result: <span style="color:#0d9488">**I decided to build this without writing a single line of manual code.**</span>
